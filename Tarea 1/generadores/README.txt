generador_desordenados: Genera datos desordenados según la cantidad de datos especificada
generador_ordenados: Genera datos desordenados según la cantidad de datos especificada, se puede hacer los ascendentes y descendentes con el mismo generador
generador_parcial: Genera datos parcialmente ordenados según la cantidad de datos especificados y un porcentaje a especificar de datos ordenados y 
el resto de datos desordenados
